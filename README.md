# Neyece 👁

> *"That's Neyece."* — the word for when a place just gets you.

**Neyece** is a warm, playful local discovery app that replaces generic search results with vibe-matched recommendations. Instead of "best restaurants near you sorted by stars," Neyece learns your taste and surfaces the spots that actually match who you are.

---

---

## The Stack
- **Frontend** — Next.js 14 + TypeScript
- **Styling** — Tailwind CSS
- **Database** — PostgreSQL (Supabase)
- **Auth** — Clerk
- **AI** — Anthropic API (Claude) for vibe matching + insights
- **Venue Data** — Google Places API + Foursquare
- **Hosting** — Vercel

---

## The Brand

The **"eye"** in Neyece is always highlighted in a coral → yellow gradient — the brand's visual wink. It represents what the app does: keeps an eye out for the next great spot, always looking out for you.

**Tagline:** *"That's Neyece."*
**Tagline system:** That's Neyece · So Neyece · Very Neyece · Stay Neyece

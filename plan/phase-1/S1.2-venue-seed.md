# S1.2 — Venue Database Seed

**Sprint:** Phase 1 · MVP Core
**Sessions:** 2
**Goal:** 500+ venues per launch city with vibe tags, stored in Supabase.

---

## Session 1 — Google Places Ingestion Script

**Focus:** Build the venue seed pipeline from Google Places API.

### Tasks
- [ ] Set up Google Places API key + configure billing
- [ ] Write seed script (`/scripts/seed-venues.ts`):
  - Query Google Places by category + city (restaurants, bars, cafes, parks, galleries, etc.)
  - Extract: name, address, neighborhood, lat/lng, category, place_id, photos, rating
  - Rate limiting: respect API quotas (max 60 QPM)
  - Deduplication: skip venues with existing `google_place_id`
  - Batch insert to Supabase `venues` table
- [ ] Define category taxonomy: ~10 categories (restaurant, bar, cafe, park, gallery, club, lounge, market, bookstore, rooftop)
- [ ] Define neighborhood taxonomy per city:
  - NYC: SoHo, Williamsburg, LES, West Village, Bushwick, etc.
  - LA: Silver Lake, Venice, DTLA, Highland Park, etc.
  - Chicago: Wicker Park, Logan Square, River North, etc.
  - Miami: Wynwood, South Beach, Little Havana, Brickell, etc.
- [ ] Run seed for NYC first as test city
- [ ] Verify: `SELECT count(*) FROM venues WHERE city = 'NYC'` → 500+

### Deliverables
- Seed script runs reliably against Google Places API
- 500+ NYC venues in Supabase with all fields populated
- Script reusable for other cities

---

## Session 2 — Vibe Tags + Image Pipeline

**Focus:** Assign vibe tags to venues, set up image storage.

### Tasks
- [ ] Define vibe tag vocabulary (~20 tags):
  - Energy: chill, lively, rowdy, intimate
  - Setting: rooftop, hidden-gem, outdoor, cozy, spacious
  - Vibe: trendy, classic, artsy, local, upscale, casual, romantic, creative
  - Scene: date-night, group-hang, solo-friendly, late-night
- [ ] Auto-assign vibe tags based on Google Places data:
  - Map categories → default tags (bar → lively, cafe → chill, etc.)
  - Use review keywords to refine (search for vibe words in reviews)
- [ ] Manually curate vibe tags for top 50 venues per city (script that opens a simple CLI for tagging)
- [ ] Image pipeline: download primary photo from Google Places → upload to Supabase Storage
- [ ] Generate thumbnail + full-size variants
- [ ] Run seed for remaining cities (LA, Chicago, Miami)
- [ ] Verify: all cities have 500+ venues, each venue has at least 2 vibe tags + 1 image

### Deliverables
- All 4 launch cities seeded with 500+ venues each
- Vibe tags assigned to every venue (auto + manual curation for top venues)
- Venue images stored in Supabase Storage

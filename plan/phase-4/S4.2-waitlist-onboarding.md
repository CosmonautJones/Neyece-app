# S4.2 — Waitlist Onboarding

**Sprint:** Phase 4 · Launch
**Sessions:** 2
**Goal:** Onboard waitlist users, collect feedback, build launch metrics dashboard.

---

## Session 1 — Invite Email Sequence

**Focus:** Design and send the 3-email onboarding sequence.

### Tasks
- [ ] Email 1 — Invite:
  - Subject: "Your city is waiting. That's Neyece."
  - Body: Exclusive invite, link to sign up, what to expect
  - Neyece voice: warm, personal, "you're one of the first"
- [ ] Email 2 — Welcome (Day 1 after signup):
  - Subject: "Welcome to Neyece. Let's find your vibe."
  - Body: Quick tips, take the vibe quiz CTA, what the score means
- [ ] Email 3 — First Discovery (Day 3):
  - Subject: "Your first Neyece spots are waiting."
  - Body: Personalized venue recommendations (top 3 by score), share CTA
- [ ] Email sending infrastructure:
  - Resend or Supabase Edge Functions + SMTP
  - HTML email templates with Neyece branding
  - Unsubscribe link
- [ ] Batch invite script: select first 50 waitlist users by city → send invite email
- [ ] Track: email sent, opened, clicked, signed up (conversion funnel)

### Deliverables
- 3 email templates written in Neyece voice
- Batch invite script working
- Email tracking in place

---

## Session 2 — Feedback + Dashboard

**Focus:** In-app feedback, user interviews, and metrics dashboard.

### Tasks
- [ ] In-app feedback prompt:
  - Trigger after 3 sessions (3 distinct app opens)
  - "How's Neyece working for you?" — 1–5 stars + optional text
  - Store in `feedback` table: user_id, rating, text, created_at
  - Dismissable, shows only once
- [ ] User interview scheduling:
  - Calendly link in profile settings: "Help us improve — book a 15min chat"
  - PostHog event: `interview_link_clicked`
- [ ] Metrics dashboard (internal, simple):
  - Page: `/admin/dashboard` (protected, admin-only)
  - DAU, WAU, MAU
  - Onboarding completion rate
  - Save rate, share rate, "That's Neyece" rate
  - Top venues by saves
  - Waitlist conversion rate
  - Neyece Score satisfaction (from feedback ratings)
- [ ] PostHog dashboards for real-time tracking
- [ ] Test: onboard 5 test users → verify full flow works

### Deliverables
- Feedback collection working
- Internal metrics dashboard
- PostHog dashboards configured

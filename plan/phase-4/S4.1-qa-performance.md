# S4.1 — QA + Performance

**Sprint:** Phase 4 · Launch
**Sessions:** 2
**Goal:** Production-ready quality, performance, and security.

---

## Session 1 — Performance Optimization

**Focus:** Core Web Vitals, load times, and mobile performance.

### Tasks
- [ ] Audit Core Web Vitals:
  - LCP (Largest Contentful Paint) < 2.5s
  - CLS (Cumulative Layout Shift) < 0.1
  - FID (First Input Delay) < 100ms
- [ ] Image optimization:
  - Next.js `<Image>` component with proper sizing/lazy loading
  - WebP format for all venue images
  - Blur placeholder for images loading
- [ ] Bundle analysis: identify and eliminate large unused dependencies
- [ ] API response times: all endpoints < 500ms
- [ ] Database query optimization: add indexes where needed
- [ ] Score computation performance: cache aggressively, avoid N+1 queries
- [ ] Supabase connection pooling configured
- [ ] Test on throttled 3G connection: app still usable

### Deliverables
- All Core Web Vitals in green
- Bundle size optimized
- API responses fast

---

## Session 2 — Security + QA

**Focus:** Security audit, error monitoring, cross-device QA.

### Tasks
- [ ] Security audit:
  - Supabase RLS policies on ALL tables (no unrestricted access)
  - API rate limiting on all public endpoints
  - Input validation (Zod) on all API routes
  - No secrets in client-side code
  - CSRF protection on mutations
  - Content Security Policy headers
- [ ] Install Sentry (`@sentry/nextjs`) for error monitoring
- [ ] Cross-device QA:
  - iOS Safari (iPhone 13+)
  - Android Chrome (Pixel, Samsung)
  - Desktop Chrome, Firefox, Safari
  - Tablet (iPad)
- [ ] Test all user flows end-to-end:
  - Sign up → quiz → feed → venue detail → save → share
  - Profile edit → taste profile → collections
  - Mood switching → feed updates
- [ ] Accessibility: keyboard navigation, screen reader basics, color contrast
- [ ] GDPR compliance:
  - Privacy policy page
  - Data deletion flow (user can delete account + all data)
  - Cookie consent (if needed for PostHog)
- [ ] Fix all discovered bugs

### Deliverables
- Security audit passed
- Sentry monitoring live
- QA across all target devices
- GDPR-compliant privacy policy + data deletion

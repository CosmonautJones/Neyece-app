# S3.3 — Share System

**Sprint:** Phase 3 · Social Layer
**Sessions:** 2
**Goal:** Branded share cards with OG images, referral tracking.

---

## Session 1 — Share Cards + OG Images

**Focus:** Dynamic OG image generation for all shareable entities.

### Tasks
- [ ] Set up `@vercel/og` for dynamic OG image generation
- [ ] Venue share card template:
  - Venue hero image as background
  - Venue name + neighborhood overlay
  - Neyece Score badge
  - User's reaction (if sharing after "That's Neyece")
  - Neyece wordmark with eye gradient
  - Coral/yellow gradient accent bar
- [ ] Collection share card template:
  - Grid of 4 venue thumbnails
  - Collection name + venue count
  - Creator's name
  - Neyece branding
- [ ] Profile share card template:
  - Avatar + name + city
  - Top 3 vibe tags
  - Neyece branding
- [ ] Share routes:
  - `/venue/[id]` → venue OG image
  - `/saved/[collectionId]` → collection OG image
  - `/profile/[username]` → profile OG image
- [ ] Test: share URLs on Twitter, iMessage, WhatsApp — verify image renders

### Deliverables
- Dynamic OG images for venues, collections, and profiles
- Images render correctly across social platforms

## Session 2 — Share Flow + Referral Tracking

**Focus:** Share UX, pre-populated copy, and attribution tracking.

### Tasks
- [ ] Share bottom sheet component:
  - "Share to..." options: Copy Link, Instagram Stories, Twitter, WhatsApp, More (Web Share API)
  - Pre-populated text: "Just found [venue] on Neyece. That's so Neyece."
  - Instagram Stories: 9:16 aspect ratio image with #SoNeyece
  - Twitter: 1.91:1 aspect ratio with hashtag
- [ ] Referral tracking:
  - Share links include referral param: `?ref=[userId]`
  - Track: share created → link clicked → signup completed
  - Attribution stored in `waitlist` or `users` table
- [ ] Create `shares` table: id, user_id, venue_id, platform, referral_code, created_at
- [ ] API route: `POST /api/shares` — log share event
- [ ] Referral stats on profile: "X people joined from your shares"
- [ ] #SoNeyece hashtag embedded in all share content automatically
- [ ] Test: share venue → click link on another device → verify referral tracked

### Deliverables
- Share flow works across platforms
- Referral attribution tracking end-to-end
- Share stats visible on profile

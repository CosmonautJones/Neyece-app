# S3.4 — Gamification

**Sprint:** Phase 3 · Social Layer
**Sessions:** 2
**Goal:** Streaks, achievements, and tier system to drive engagement.

---

## Session 1 — Streaks + Achievements

**Focus:** Build streak tracking and achievement system.

### Tasks
- [ ] Create `streaks` table: user_id, current_streak, longest_streak, last_active_week, updated_at
- [ ] Streak logic (Supabase function):
  - "Active week" = user discovered (detail viewed) at least 1 new venue
  - Consecutive active weeks = streak count
  - Streak resets if a week is missed
  - Run check weekly (cron or on-login)
- [ ] Create `achievements` table: id, user_id, achievement_type, unlocked_at
- [ ] Achievement definitions:
  - "First Neyece" — first "That's Neyece" tap
  - "Explorer" — viewed 10 venue details
  - "Scout" — saved 10 venues
  - "Insider" — shared 5 venues
  - "Tastemaker" — 5-week streak
  - "Legend" — 20-week streak + 50 saves + 20 shares
- [ ] Achievement check function: run on signal events, unlock when criteria met
- [ ] Achievement unlock notification: toast/modal with animation
- [ ] Streak counter displayed in home screen header (flame icon + number)
- [ ] Test: simulate user actions → verify achievements unlock correctly

### Deliverables
- Streak tracking working with weekly cadence
- 6 achievements defined and auto-unlocking
- Streak counter on home screen

## Session 2 — Tier System + Profile Integration

**Focus:** Scout tier system and displaying gamification on profile.

### Tasks
- [ ] Tier system based on cumulative activity:
  - **Regular** — default (0 points)
  - **Scout** — 50 points (saves=2pts, shares=3pts, streaks=5pts/week)
  - **Tastemaker** — 200 points
  - **Legend** — 500 points
- [ ] Tier badge on profile page (colored icon)
- [ ] Tier badge on public profile and share cards
- [ ] Achievements section on profile: grid of unlocked achievements with icons
  - Locked achievements shown as greyed-out silhouettes
- [ ] Streak display on profile: current streak + longest streak
- [ ] Progress bar to next tier on profile
- [ ] Leaderboard (future consideration, not MVP — just plan the table)
- [ ] Lightweight animations: tier-up celebration, achievement unlock
- [ ] Test: accumulate points → verify tier progression → check profile display

### Deliverables
- 4-tier system with point accumulation
- Tier badges on profile and share cards
- Achievements displayed on profile page
